## üéØ Overview

**AXIS Studios** is not just a collection of files; it's a comprehensive, semi-automated system designed to streamline the creation of high-quality, premium Shopify themes. It acts as a "50-person agency in a single brain," guiding you from market research and concept creation all the way to a successful Theme Store submission.

This system is built to enforce best practices in performance, accessibility, design, and security, ensuring that every theme created not only meets but exceeds Shopify's rigorous standards.

## ‚ú® Core Concepts

The system operates on a few core philosophies:

1.  **Performance by Default**: Every decision and generated piece of code prioritizes speed and a high Lighthouse score.
2.  **Accessibility is Law**: We adhere strictly to WCAG 2.1 AA standards to ensure themes are usable by everyone.
3.  **Uniqueness Sells**: Built-in checks and creative guidance prevent the creation of generic, "Dawn-like" themes.
4.  **AI as a Partner**: The system automates tedious tasks, allowing you, the visionary, to focus on strategy and creative direction.
5.  **Continuous Learning**: With the `Memory Log`, the system learns from its mistakes and gets smarter with every project.

## üöÄ Getting Started

To start building your first theme, use the `START NEW THEME` command. This will kick off the automated market research and strategy phase.

**Example:**

```bash
> START NEW THEME "Zenith" --niche=furniture
```

The AI will then analyze the competition in the "furniture" niche and propose a unique concept for your theme, "Zenith".

## ü§ñ Main Commands

You interact with the AXIS system using a simple, command-like syntax. Here are the most important commands defined in `axis_brain_v10.0_GOD_MODE.json`:

| Command                                             | Description                                                               |
| --------------------------------------------------- | ------------------------------------------------------------------------- |
| `> START NEW THEME "[NAME]" --niche=[niche]`        | Begins a new theme project by performing market analysis.                 |
| `> APPROVED. GENERATE CORE`                         | Approves the AI's concept and generates the theme's file structure.     |
| `> BUILD SECTION "[name]"`                          | Generates the complete code for a specific theme section.                 |
| `> GENERATE DEMO_DATA --niche=[niche]`              | Creates realistic demo content (products, blogs) for your demo store.     |
| `> RUN FINAL AUDIT`                                 | Executes all automated tests (Performance, A11Y, Security).             |
| `> PACKAGE FOR SUBMISSION --theme=[name] --ver=[v]` | Packages the theme into a `submission.zip` for the Shopify Theme Store. |
| `@full-context [your query]`                        | Forces the AI to load all system files for complex, multi-faceted tasks.  |
