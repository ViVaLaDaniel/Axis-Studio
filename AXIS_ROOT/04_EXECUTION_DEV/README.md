# 04_EXECUTION_DEV

Эта папка содержит ключевые файлы, определяющие стандарты разработки и дизайн-токены для AXIS OS.

## Содержание:

- [design_tokens.json](./design_tokens.json)
  - Содержит все дизайн-токены (цвета, типографика, отступы, анимации, радиусы скругления, тени, компоненты), которые научно связаны с AXIS OS и Living Styleguide.
- [CODING_STANDARDS.md](./CODING_STANDARDS.md)
  - "Единая техническая конституция" для разработчиков AXIS OS, охватывающая стандарты кодирования для Liquid, JavaScript, CSS, а также правила именования, файловой системы, доступности, безопасности, бюджета производительности, сигналов событий, архитектуры секций и сниппетов, и поведения DevOps.
- [CODE_SNIPPETS_LIBRARY.md](./CODE_SNIPPETS_LIBRARY.md)
  - Библиотека переиспользуемых сниппетов кода, соответствующих стандартам AXIS OS для Liquid, JavaScript и CSS.
