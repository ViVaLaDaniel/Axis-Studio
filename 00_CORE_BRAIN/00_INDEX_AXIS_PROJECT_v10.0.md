# Master Index: AXIS OS v10.5

This document serves as the master index and high-level map for the entire AXIS OS project. It outlines the core architectural components and defines the key roles involved in its operation.

---

## 1. Module Architecture

The AXIS OS is built upon a modular architecture where each numbered directory represents a core system component with a distinct responsibility.

- **[00_CORE_BRAIN](./)**
  - **Role:** The Central Nervous System.
  - **Description:** Manages AI decision-making, context, memory, and orchestrates the entire workflow. This is the "mind" of the AI.

- **[01_FOUNDATION_RULES](../01_FOUNDATION_RULES/)**
  - **Role:** The Constitution.
  - **Description:** Contains the immutable laws of the project, including technical specifications, design principles, and priority rules.

- **[02_STRATEGY_CREATIVE](../02_STRATEGY_CREATIVE/)**
  - **Role:** The Creative & Scientific Hub.
  - **Description:** Leverages UX research, psychological principles, and market data to generate unique, high-conversion theme strategies.

- **[03_ARCHITECTURE](../03_ARCHITECTURE/)**
  - **Role:** The Architectural Bureau.
  - **Description:** Designs the theme's structural blueprint, including component hierarchies, data flows, and architectural patterns.

- **[04_EXECUTION_DEV](../04_EXECUTION_DEV/)**
  - **Role:** The Engineering Floor.
  - **Description:** Contains the tools and standards for writing actual code: snippets, coding standards, and design tokens.

- **[05_QUALITY_SECURITY](../05_QUALITY_SECURITY/)**
  - **Role:** Quality Assurance & Security.
  - **Description:** Manages all testing protocols, security scans, and accessibility checks to ensure a high-quality, robust product.

- **[06_CI_CD_RELEASE](../06_CI_CD_RELEASE/)**
  - **Role:** The Release & Deployment Department.
  - **Description:** Handles versioning, build processes, demo content generation, and packaging the final theme for the Shopify Theme Store.

- **[07_TEAM_DOCS](../07_TEAM_DOCS/)**
  - **Role:** The Project Library & HR.
  - **Description:** Stores all team-related documentation, role definitions, product playbooks, and workflow guides.
